## Gym Membership Data Analysis

This project analyses a gym to uncover patterns in their attendance, membership types and what type of workout types they prefer. Using SQL for data extraction, Excel for exploration and PowerBI for interactive dashboards.

### Problem Statement

The objective of this project is to understand member engagement and identify trends in attendance, preferences, workout types, and age groups who go to the gym. This analysis helps highlight opportunities to improve on retention, and focus on certain demographics.

### Key Insights

- Identified differences in attendance patterns among membership types
- Discovered which workout types are most popular across different age groups
- Highlight peak visit days and potential capacity opportunities
- Reveal behavioural trends that could impact customer retention strategies

### Tools & Technologies

- SQL (data cleaning, querying, aggregation)
- Excel (data exploration)
- Power BI (interactive dashboard and data visualization)

### Conclusion

The analysis reveals that gym attendance is heavily concentrated among senior members, with this group consistently showing the highest participation across all categories, including membership types and workout preferences. Compared to other age groups, seniors demonstrate stronger engagement and more regular attendance patterns, making them the most active segment of the gym’s customer base.

This trend suggests that the gym’s current offerings may be particularly well-suited to senior members, while younger age groups appear to be less engaged. As a result, there is an opportunity for the gym to develop targeted strategies to increase participation among underrepresented groups.

Overall, these insights highlight the importance of understanding member demographics in driving attendance, and demonstrate how data can be used to improve engagement, optimize offerings, and support more informed, data-driven business decisions.
